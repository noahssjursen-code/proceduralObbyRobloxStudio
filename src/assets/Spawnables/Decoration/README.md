# Decoration Models

Drop **decorative models** here in Roblox Studio.

## 📍 Location in Studio
`ReplicatedStorage > Assets > Spawnables > Decoration`

## 🎨 What to Put Here
Non-gameplay decorations that add atmosphere:
- Floating crystals
- Glowing orbs
- Ancient runes
- Particle clouds
- Ambient lights
- Architecture details
- Anything atmospheric!

## 📐 Sizing
Base size: ~0.5-1 stud
These scatter around the path for visual interest.

## ⚙️ Visual Effects
Decorations should be **non-collidable** and purely visual:
- Particle emitters
- Point lights
- Transparency effects
- No collision!

The system will set `CanCollide = false` automatically.

The system will automatically find and use these models! 🎮
